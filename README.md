# AI Code Review Assistant

> Automatically reviews GitHub Pull Requests using Llama 3 AI and posts
> structured feedback as PR comments — covering bugs, security, style,
> performance, and suggestions.

## What It Does

When a developer opens a Pull Request, this tool automatically:

- Detects the PR via GitHub webhook
- Fetches all changed code using GitHub API
- Sends the diff to Llama 3 (Groq) for intelligent analysis
- Posts a structured review comment back to the PR in seconds

No manual review triggering — fully automated.

## How It Works

```
Developer opens PR
        ↓
GitHub fires webhook → Flask server receives it
        ↓
GitHub API fetches changed files + diff
        ↓
Diff sent to Llama 3 (Groq) for analysis
        ↓
Structured review posted as PR comment
```

## Run It Locally

**1. Clone the repo**
```bash
git clone https://github.com/Muzzamil-8/ai-code-reviewer
cd ai-code-reviewer
```

**2. Install dependencies**
```bash
pip install -r requirements.txt
```

**3. Create `.env` file**
```
GITHUB_TOKEN=your_github_token
GROQ_API_KEY=your_groq_key
GITHUB_OWNER=your_github_username
GITHUB_REPO=your_repo_name
```

**4. Run the server**
```bash
python app.py
```

**5. Expose locally and add webhook**
```
Expose:   ssh -R 80:localhost:5000 nokey@localhost.run
Webhook:  GitHub repo → Settings → Webhooks → Add webhook
URL:      https://your-url/webhook
Events:   Pull requests only
```
